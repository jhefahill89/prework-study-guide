HTML

  • HTML is the standard markup language for creating webpages.
  • The head element contains information about the web page.
  • The body element represents the visible content shown to the user.
  • HTML describes the structure of a webpage.
  • HTML consists of a series of elements.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicate how much space we want around the content inside an element.
  • When a developer wants to style an element directly in the HTML file, they use inline CSS.

Git

  • git status: checks what branch we are currently working on
  • git checkout -b branch name: creates a new branch and switches to it

javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript is a high-level dynamic programming language that allows users to interact with websites.
  • In web development, JavaScript is the tool that takes a static HTML/CSS webpage and makes it interactive.
  • The word console refers to a test environment that developers use to check out their code.
